## [1.0.0] - 2024-10-12

### المرحلة الأولى (MVP) - مكتملة ✅

#### إضافات
- نظام تسجيل المستخدمين والسائقين
- نظام المصادقة باستخدام Laravel Sanctum
- نظام OTP للتحقق من رقم الهاتف
- إدارة المدن والمسارات
- نظام حجز الرحلات
- نظام توصيل الطرود
- لوحة إدارة Filament
- واجهات المستخدم بـ React + Inertia.js
- API RESTful كامل
- نظام الأرباح والعمولات
- نظام التقييمات
- محفظة إلكترونية
- بيانات تجريبية شاملة

#### قاعدة البيانات
- جدول users

- صفحة الأرباح للسائق
